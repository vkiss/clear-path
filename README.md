# clear-path
[![npm version](https://badge.fury.io/js/clear-path.svg)](https://badge.fury.io/js/clear-path)

A node library for deleting a path if exists.

## Instalation

```bash
npm i clear-path --save-dev
```

## How (in JS)
```js
const clearPath = require( "clear-path" );

clearPath("./dist");

```
